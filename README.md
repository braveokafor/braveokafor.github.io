### What I work on

- **Control plane architecture** - designing the management layers that provision, orchestrate, and expose infrastructure as a reliable platform API
- **Distributed systems** - lifecycle state machines, failure recovery, consistency under partial failures, multi-cloud behavioural differences
- **Kubernetes at depth** - cluster operations, CNI networking, custom controllers and admission webhooks, workload scheduling (CNCF Kubestronaut)
- **Storage systems** - tiering, durability design, access pattern-driven optimisation, cost efficiency at scale
